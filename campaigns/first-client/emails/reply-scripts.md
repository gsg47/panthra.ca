# Reply scripts

Respond the same day when possible. Keep replies human — short paragraphs, one next step.

---

## 1) Interested — “Sure, let’s talk”

```
Great — thanks {{FirstName}}.

Here’s a Calendly/booking link: {{BookingLink}}
Or tell me two windows that work this week and I’ll lock one in.

Before the call, useful (optional) context:
• Rough headcount + whether you’re mostly on Microsoft 365
• Any recent scare or insurance / client questionnaire driving this

Talk soon,
Gurshaan
```

If no booking link yet: propose two concrete MT slots.

---

## 2) Soft interest — “Send more info”

```
Happy to.

Short version: 30 minutes, screenshare optional, focused on email/identities, backups, and a simple incident first-hour plan for {{Company}}. You leave with a plain-language priority list — not a 40-page PDF.

If it’s useful, we can follow with a paid assessment or monitoring starter. No obligation either way.

Want me to hold a time, or would you rather skim a one-pager first? I can send that today.
```

---

## 3) Not the right person

```
Appreciate the redirect — who should I speak with on security / IT / operations?

If you can CC them (or share an email), I’ll keep it brief and specific to {{Company}}.
```

---

## 4) “We already have IT / an MSP”

```
Makes sense — a lot of good MSPs cover day-to-day IT.

Where PANTHRA usually helps is the security layer on top: phishing-resistant email/identity controls, monitoring that someone actually watches, and a practiced response path when something slips through.

If a second set of eyes on that slice would be useful, I’m happy to do a pulse check alongside your current provider — not replace them.
```

---

## 5) “Not now” / timing

```
Totally fair. I’ll close the loop on my side.

If you want this later, reply “pulse check” anytime and I’ll send two time options. Wishing you a quiet year.
```

Mark tracker `status=nurture` and set a follow-up date ~60–90 days out.

---

## 6) Price / “How much?”

```
Discovery call is free.

If we move to paid work after that, it depends on scope (assessment vs monitoring vs hardening). Most first engagements for teams your size start in a clear fixed range — I’ll give numbers on the call once I understand {{Company}}’s setup, so you’re not buying a black box.
```

---

## 7) Skeptical / “Is this a sales blast?”

```
Fair question. This note was written for {{Company}} because {{HookOneLiner}} — not a mass mail merge.

If it’s not relevant, say the word and I won’t follow up. If a short pulse check would still be useful, I’m easy to schedule.
```

---

## 8) Unsubscribe / “Stop emailing”

```
Understood — you’re off my list. Sorry for the noise.
```

Mark `status=unsubscribed`. Do not send Email 2/3.

---

```
Thanks again for the time today, {{FirstName}}.

Attached / below: the 3 priorities we discussed for {{Company}}, in plain language.
If you want help implementing #1 first, I can send a fixed-scope proposal by {{Date}}.

Either way, glad we connected.
```
